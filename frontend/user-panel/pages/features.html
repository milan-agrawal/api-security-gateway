Intelligent Threat Protection

Automated defense against known and emerging attack vectors

Every request passing through the gateway is automatically inspected for malicious intent. Your APIs are shielded from a wide range of threats — with no configuration required on your part.

Real-Time Scanning Every request is inspected before reaching your API — headers, parameters, and payloads included.
Automatic Blocking Malicious traffic is stopped at the edge and never forwarded to your backend.
Multi-Vector Defense Protection spans across request headers, query strings, body content, and URL paths.
Zero Configuration All protections are active by default the moment you route traffic through the gateway.
Threat protection is always active — there are no toggles or settings to manage. Your APIs are secured from the first request.

Adaptive Security

Self-learning system that evolves with your traffic patterns

Beyond static protections, the gateway employs adaptive analysis that understands your normal traffic patterns and identifies deviations. New and previously unseen attack methods can be caught — even without a predefined signature.

Behavioral Analysis Understands what normal looks like for your API traffic and flags deviations.
Anomaly Detection Identifies unusual patterns that static protections alone might miss.
Continuous Learning Protection gets smarter automatically as more traffic flows through the system.
Zero-Day Coverage Novel attack patterns are detected even without prior knowledge of the specific threat.
Adaptive security works alongside threat protection to provide layered defense — multiple independent systems evaluate every request.

Real-Time Decision Engine

Millisecond security verdicts on every request

Every request receives a security verdict almost instantly. Multiple analysis layers work together to score each request for risk — and malicious traffic is blocked before it ever reaches your API.

Instant Verdicts Security decisions are made in under 50 milliseconds — no noticeable latency added.
Risk Scoring Every request receives a risk assessment combining signals from multiple analysis layers.
Correlated Analysis Independent security signals are combined for higher accuracy and fewer false positives.
Full Audit Trail Every decision is logged and traceable — complete transparency into allow/block outcomes.
Allowed Clean — forwarded to your API
Blocked Malicious — rejected immediately
Flagged Suspicious — logged for review
If a legitimate request is blocked, you can submit an appeal from the Appeal Portal in your dashboard.

Smart Rate Management

Fair usage enforcement and automatic abuse prevention

To maintain service quality and prevent abuse, the gateway enforces intelligent rate limits on a per-key basis. This ensures fair access for all consumers while protecting your APIs from being overwhelmed.

Per-Key Quotas Each API key has its own request allowance within a rolling time window.
Automatic Throttling Approaching your limit triggers gradual slowdown before any hard blocks.
Abuse Prevention Automated detection and response to abnormal usage spikes and scraping attempts.
Fair Scheduling Ensures equitable access across all API consumers, even during high-traffic periods.
View your current rate limits and usage on the Usage Plans page. Limits vary by plan tier.

Account & Access Security

Enterprise-grade protection for your account and credentials

Your account is protected by multiple security layers designed to prevent unauthorized access. From encrypted credentials to multi-factor authentication, every aspect of account security is built-in.

Multi-Factor Authentication Add a second layer of verification with authenticator-app-based two-factor authentication.
Secure Sessions Sessions are encrypted and time-limited; expired sessions are automatically invalidated.
Credential Protection Passwords are hashed and never stored in readable form. Recovery flows are secure by design.
Session Management View all active sessions and revoke any of them instantly from your Profile page.
We strongly recommend enabling Two-Factor Authentication from your Profile page — it's the single most effective step to secure your account.

Visibility & Insights

Complete transparency into your API traffic and security events

The gateway provides comprehensive logging and analytics so you always know what's happening with your API traffic. Every request, every decision, and every security event is captured and accessible.

Request Logging Every request is recorded with full metadata — method, path, status, latency, and outcome.
Security Events Real-time records of blocked, flagged, and suspicious requests with detailed context.
Request Tracing Each request gets a unique ID for complete lifecycle tracking and support reference.
Appeal System Believe a request was incorrectly blocked? Submit an appeal directly from your dashboard.
All logs and security events are accessible from the Security Logs page in your dashboard.
